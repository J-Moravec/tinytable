IttyBittyTable is easy to use

# library(IttyBittyTable)
pkgload::load_all()

x <- mtcars[1:4, 1:5]

IttyBittyTable(x)

Output formats

IttyBittyTable can produce tables in HTML, Markdown, or LaTeX (PDF) format. To choose, we use the output argument:

IttyBittyTable(x, output = "html")
IttyBittyTable(x, output = "latex")
IttyBittyTable(x, output = "markdown")

When calling IttyBittyTable from a Quarto or Rmarkdown document, IttyBittyTable detects the output format automatically and generates an HTML or LaTeX table as appropriate. This means that we do not need to explicitly specify the output format.

Style

The style() function allows us to apply visual styles to our table. This includes customizing features such as:

  • Text color
  • Background color
  • Widths
  • Heights
  • Alignment
  • Text Wrapping
  • Column and Row Spacing
  • Cell Merging
  • Multi-row or column spans
  • Border Styling
  • Font Styling
  • Header Customization

The main arguments of the style() function are rather self-explanatory:

  • i row numbers: integer vector or NULL to style all rows.
  • j column numbers: integer vector or NULL to style all columns.
  • color: text color
  • background: background color
  • bold: bold text
  • italic: bold text
  • align: horizontal alignment

In addition, style() accepts two more arguments which allow unlimited possibilities for customizing every possible aspect of your tables in HTML or LaTeX (PDF):

  • latex = latexOptions() for tabularray
  • html = htmlOptions() for Bootstrap

We discuss these extra arguments near the end of this page.

Cells

To style individual cells, we use the style_cell() function. The first two arguments (i and j) identify the positions of the cells of interest, by row and column numbers respectively. To style a cell in the 2nd row and 3rd column, we can do:

IttyBittyTable(x) |>
  style(
    i = 2,
    j = 3,
    background = "black",
    color = "white",
    bold = TRUE,
    italic = TRUE)
IttyBittyTableohm0emb8c3yn59o6jfe5

21 6 160 110 3.9
21 6 160 110 3.9

21.4 6 258 110 3.08

The i and j accept vectors of integers to modify several cells at once:

IttyBittyTable(x) |> style(2, c(1, 3), background = "olive")

HTML customization

Warning: The html argument is only used for HTML tables, and it is ignored when IttyBittyTable creates LaTeX tables for PDF files. You can skip this section if you are reading the PDF version of this tutorial.

Themes

The Bootstrap framework provides a number of built-in themes to style tables. To use them, we call htmlOptions() with the class argument, and we specify the Bootstrap class. A list of available Bootstrap classes can be found here: https://getbootstrap.com/docs/5.3/content/tables/

For example, to produce a “dark” table, we use the table-dark class:

IttyBittyTable(x, html = htmlOptions(class = "table table-dark"))
IttyBittyTableyowj8rg2jujpqqqt1n23
mpg cyl disp hp drat
21 6 160 110 3.9
21 6 160 110 3.9
22.8 4 108 93 3.85
21.4 6 258 110 3.08

We can also combine several Bootstrap classes. Here, we get a “striped” table with the “warning” color:

IttyBittyTable(x, html = htmlOptions(class = "table table-striped table-warning"))

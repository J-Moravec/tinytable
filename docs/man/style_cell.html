Style Cells in a tinytable

Description

This function applies styling to specified parts of a table created by the tinytable() function of the tinytable package.

The arguments described below allow users to customize text style and color. The latex and html arguments allow deep customization using the tabularray package for LaTeX, and the Bootstrap framework for HTML. For details, see:

  • ?latexOptions

  • ?htmlOptions

Usage

style_cell(
  x,
  i = NULL,
  j = NULL,
  color = NULL,
  background = NULL,
  bold = FALSE,
  italic = FALSE,
  latex = latexOptions(),
  html = htmlOptions()
)

Arguments

x A table created by the tinytable() function.
i Integer vector indicating row positions. If NULL, applies to all rows.
j Integer vector indicating column positions. If NULL, applies to all columns.
color String specifying the text color. If NULL, no color is applied.
background String specifying the background color. If NULL, no background is applied.
bold Logical; if TRUE, applies bold styling to the text.
italic Logical; if TRUE, applies italic styling to the text.
latex sytling options for the tabularray framework for LaTeX. See ?latexOptions for details.
html styling options for the Bootstrap framework for HTML. See ?htmlOptions for details.

Value

Returns the modified table object with the specified cell styles applied.
